# Instagram-social-media-data-analysis
Analyzed an Instagram analytics dataset sourced from Kaggle using Python and Pandas to explore engagement patterns, content performance, and follower growth through data cleaning and visualization.
I began by sourcing an Instagram analytics dataset from Kaggle and uploading it into the notebook environment. The first step involved understanding the structure of the dataset and identifying key engagement metrics such as likes, comments, shares, saves, reach, and followers gained. I then performed data cleaning by handling date formats, checking for missing or invalid values, and removing unnecessary columns to prepare the data for analysis.

Once the data was cleaned, I conducted exploratory data analysis using statistics and visualizations to understand distributions, engagement patterns, and relationships between variables. Bar charts, histograms, boxplots, and scatter plots were used to compare performance across media types, traffic sources, and content categories.
Solution: Using pandas and visualization libraries such as matplotlib and seaborn, I analyzed how different content formats and traffic sources influence engagement and follower growth. The analysis revealed that engagement levels were relatively consistent across media types and traffic sources, with slight variations that highlight how content discovery mechanisms contribute to audience growth.

The project successfully met its objective by transforming raw Instagram analytics data into meaningful insights through structured analysis and clear visual communication.
